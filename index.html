JMI Dynamic Weighted CGPA Calculator Batch (2021-2025)

Enter GPA for completed semesters only. The CGPA will be calculated based on the data provided, using yearly weights.

Dynamic Calculation Logic:

The calculation adapts based on the semesters you provide GPA values for.

Weights: Year 1 (S1/S2) = 25%, Year 2 (S3/S4) = 50%, Year 3 (S5/S6) = 75%, Year 4 (S7/S8) = 100%

Numerator = Sum of (Weight * Entered GPA) for each entered semester.
Denominator = Sum of (Weight * 10) for each entered semester.
Final CGPA = (Numerator / Denominator) * 10

Example: If only S1, S2, S3 entered: Num = 0.25*S1 + 0.25*S2 + 0.50*S3. Denom = 0.25*10 + 0.25*10 + 0.50*10 = 10. CGPA = (Num/10)*10.
